# Sistema de Cálculo de Seguro e IPVA Automotivo

Este projeto é uma aplicação web desenvolvida em JavaScript para cálculo automático de seguro, IPVA e valor final de veículos com base em dados fornecidos pelo usuário.

---

## Funcionalidades

- Cadastro de veículos
- Cálculo automático de seguro (10% do valor do veículo)
- Cálculo de IPVA conforme tipo de combustível
- Isenção de IPVA para veículos com mais de 20 anos
- Cálculo do valor final do veículo
- Listagem dinâmica de veículos cadastrados
- Limpeza da lista de veículos

---

## Regras de cálculo

### Seguro
O seguro corresponde a 10% do valor do veículo.
